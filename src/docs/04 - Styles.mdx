# 4. Styling Guide

Organized has a sophisticated design system with full support for **8 themes** (4 colors × light/dark). All tokens live in `src/global/global.css` and are applied via a `data-theme` attribute on the root element.

## CSS Variables

**Never hardcode `#hex` or `rgb()` values.** Always use CSS variables so the app remains fully theme-compliant.

## MUI Overrides

We use MUI v7 as a base layer. Override component styles with CSS variables via the `sx` prop — never use MUI's `palette` system directly:

```tsx
<Box sx={{
  backgroundColor: 'var(--white)',
  borderRadius: 'var(--radius-l)',
  boxShadow: 'var(--small-card-shadow)',
  border: '1px solid var(--grey-200)',
}}>
```
